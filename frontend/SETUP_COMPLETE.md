## 🚀 **How to Run**

### Quick Setup (Windows)
```cmd
cd frontend
setup.bat
```

### Manual Setup
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

### Start Both Services
```bash
# Terminal 1 - Backend
python -m src.main serve

# Terminal 2 - Frontend  
cd frontend
npm run dev
```

Then open:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
